This repository contains all files and data for a minimal testcase with two domains centered at Jungfraujoch (JFJ) station.
WPSfiles are meant to generate the met_em* files from scratch if WPS needs to be tested.
WRFfiles are meant to test changes in WRF only. SNOWPACK files *.sno are not loaded here but they can be easily generated after running .real.exe 
and the .py script.
Results from the atmospheric weather station are also present here for comparison.
